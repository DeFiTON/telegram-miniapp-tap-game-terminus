# Telegram Mini App Tap Game — Terminus

> A tap-to-earn clicker game built as a Telegram Mini App on the TON blockchain. Tap to farm coins, climb leagues, complete daily tasks, join clans and invite friends.

[![Website](https://img.shields.io/badge/Play-wm--mariupol.com-1f6feb)](https://wm-mariupol.com)

## What it is

This is a **Telegram Mini App tap game** — a clicker / tap-to-earn game that runs directly inside Telegram with no separate app to install. Players tap to farm in-game coins, level up through leagues, complete tasks, join clans and earn rewards by inviting friends. The game is integrated with the **TON blockchain** and TON Connect wallets.

It is built for **casual players** who enjoy tap-to-earn games and for the **Telegram crypto-gaming** audience.

## Key features

- **Tap-to-earn gameplay** — tap to collect in-game coins.
- **Leagues and levels** — progress through league tiers and coin-progress levels.
- **Daily tasks** — complete tasks for extra rewards.
- **Clans** — join or create clans and play together.
- **Referral / invite system** — invite friends with an invite code and earn rewards.
- **Premium** — optional premium features.
- **TON integration** — works with TON Connect wallets; on-chain elements via the TON network.
- **Runs inside Telegram** — a full Mini App experience, no install required.

## How to play

1. Open the game inside Telegram or visit [wm-mariupol.com](https://wm-mariupol.com).
2. Tap the coin to start farming in-game coins.
3. Complete daily tasks for bonus rewards.
4. Climb the league tiers as your balance grows.
5. Join a clan and invite friends with your invite link.
